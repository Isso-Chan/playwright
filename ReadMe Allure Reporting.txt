In order to get Allure Reports;
1. Watch this video and apply all-> https://www.youtube.com/watch?v=rLwMK4D2aPI
2. If you have just created allure reports by means of CMD window, get cursor by clicking on Ctrl+C (2 times)
3. Remove "allure-results" folder under project root
4. Run test cases which are defined in xml runner-file
5. Open cmd and move to project folder (using cd)
6. Type -> "allure serve allure-results" and click enter
