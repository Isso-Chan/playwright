If you have just created allure reports by means of CMD window, get cursor by clicking on Ctrl+C (2 times) 3. Remove "allure-results" folder under project root 4. Reports are generated in 2 ways; a. In TestRunner class (1) Delete "allure-results" folder under project root (if there is any) (2) Just run TestRunner class (3) Then reports folder will be generated under PROJECT root (not in target) as allure-results (4) Type -> "allure serve allure-results" and click enter (5) Report will be opened in your default browser (6) Clear this folder MANUALLY ("mvn clean" do NOT delete them!) b. With maven command (1) Type-> mvn clean (2) Just -> "mvn verify" in the terminal (with cmd, first move to your project folder!) (3) Then reports folder will be generated under TARGET root as allure-results (4) Type -> "allure serve target/allure-results" and click enter (5) Report will be opened in your default browser
